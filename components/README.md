# Components

This directory contains reusable React components for the Fullpreneur Dashboard.

## Sidebar.tsx

Discord-style navigation sidebar with:
- Main navigation items (Home, CRM, Calendar, Creative Space, The Vault)
- Revenue streams list in footer
- Active route highlighting
- Clean, nested structure (no hashtags)

## Future Components

Potential additions:
- LeadCard.tsx - Individual lead display
- EventCard.tsx - Calendar event component
- FulfillmentCard.tsx - Fulfillment tracking item
- RevenueStreamWidget.tsx - Revenue stream summary
